3.2 API

Common Lisp Function: (add-template-directorydirectory &optional (template-store *current-store*))

Adds DIRECTORY to the search path of the TEMPLATE-STORE

Common Lisp Generic: (compile-templatecompiler name &optional error-p)

Provides a hook to customize template compilation.


Specializers:

  • * (toplevel-compiler common-lisp:t)
  • * (compiler common-lisp:t)
Common Lisp Function: (compile-template*name)

Compiles template NAME with compiler in `CURRENT-COMPILER'

Common Lisp Function: (render-template*template &optional stream &rest *template-arguments*)

Render TEMPLATE into STREAM passing `TEMPLATE-ARGUMENTS'
